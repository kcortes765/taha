# Reports WS2

Carpeta para devolver evidencia desde WS2 hacia el APOS principal.

Nombres sugeridos:

```text
WS2_REPORTE_MODELO_ED1_YYYYMMDD_HHMM.md
WS2_APOS_DELTA_YYYYMMDD_HHMM.md
```

No guardar aqui archivos `.EDB` salvo decision explicita. Para modelos ETABS, usar backups locales en `HECRAS2`.
