# Categorías de gráficos PNG/SVG

Paquete vigente: `WS2_VISUALES_TIPO_DIOS_20260509_2320`.

Todo este paquete vive dentro de `HECRAS2\prog2`, que queda como base ordenada para los entregables visuales del informe.

Cada carpeta incluye el PNG de alta resolución y el SVG editable de la misma figura.
