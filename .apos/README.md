# APOS

Memoria local del proyecto `1° Sem. 2026 - UCN`.

Esta carpeta contiene estado vivo, decisiones, bitacora, fuentes, riesgos,
research y evidencia. No es un dump del chat.

Reglas APOS-X:
- Leer `CONTEXT_POLICY.md`, `INDEX.md`, `STATUS.md`, `HANDOFF.md` y `PLAN.md` para retomar.
- No inventar contexto.
- Clasificar hechos, decisiones, inferencias, pendientes y riesgos.
- Agregar, no reescribir, `DECISIONS.md`, `JOURNAL.md`, `SOURCES.md` y `RESEARCH_LOG.md`.
- Usar `apos-run` para ejecuciones productivas, grandes, destructivas o ambiguas.
